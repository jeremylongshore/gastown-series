# Gastown Series

Multi-agent production pipeline for the Gastown series.

## Structure

```
gastown-series/
  skills/          # Agent role definitions
  bible/           # Canonical world, characters, and episode log (append-only)
  output/          # All deliverables
    scripts/       # Script drafts and finals
    assets/        # Character sheets, mood boards, reference material
    episodes/      # Assembled episode packages
  AGENTS.md        # Agent directives (read first)
```

## Getting Started

1. Read `AGENTS.md` for agent rules
2. Read the relevant skill file(s) in `skills/`
3. Review `bible/` for current canon
4. Track work with beads, output to `output/`
