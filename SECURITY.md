# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the production pipeline scripts or tooling, please report it by opening a private issue or contacting the maintainers directly.

Do not open public issues for security vulnerabilities.

## Scope

This policy covers the technical tooling in this repository:

- Voice generation scripts (`output/audio/`)
- Asset generation scripts (`output/assets/`)
- Assembly pipelines (`output/`)

Creative content (bible, scripts, episodes) is not in scope for security reports.
